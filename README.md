# Chartered Accountant Website

A professional website for a Chartered Accountant (CA) firm built with pure HTML, CSS, and JavaScript.

## Overview

This is a clean, responsive business website designed for a Chartered Accountant or accounting firm. It provides clients with information about services, legal compliance pages, and contact details.

## Features

- **Services Section**: Overview of CA services (auditing, tax filing, compliance, etc.)
- **Legal Page**: Terms, privacy policy, and legal information
- **Responsive Design**: Mobile-friendly layout
- **Contact Form**: Easy client inquiry form
- **Professional UI**: Clean and trustworthy business design

## Project Structure

```
chartered-accountant-website/
├── images/          # Website images and assets
├── index.html       # Main homepage
├── legal.html       # Legal/Terms page
├── script.js        # JavaScript functionality
└── style.css        # Stylesheet
```

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

```bash
git clone https://github.com/Icebear-cyber/chartered-accountant-website.git
cd chartered-accountant-website
```

Open `index.html` in your browser to view the website.
